# Galvanize Eats
```
Galvanize eats is a simple food delivery web app.  Users can sleect items from the menu brought in from 
the API, select the quantity and tip amount, and post the order back to the API.
```

## Deployed Address
```
g-eats102.surge.sh
```

## Tech Used
```
- Vanilla JS
- HTML5
- CSS3
- FETCH
```
